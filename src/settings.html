Settings

Daily/weekly task files (YYYY-MM-DD.json) are written here. Point this at a Google Drive, iCloud, or Dropbox folder for automatic backup and restore. Each day is a separate file, so syncing is fast and conflicts are rare.
Daily files keep each day small and make cloud sync incremental. Weekly files reduce file count if you prefer.
When off, the app only shows in the menu bar / system tray. Restart the app for this to take effect on macOS.
